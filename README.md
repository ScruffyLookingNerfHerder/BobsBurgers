# Project Name - Bob's Burgers

## Description
A coder's take on the popular TV show "Bob's Burgers," this app allows you to create, post, and devour your own pun burgers. It is deployed to heroku.

## Key Features
- Create Burgers
- Devour Burgers


## Tech Used
- JS
- HTML
- Handlebars.js
- MySQL
- JawsDB
- CSS
